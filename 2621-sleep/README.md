# 2621. Sleep (Pausa Assíncrona)

### Descrição do Problema
Dado um número inteiro positivo `millis`, o objetivo é criar uma função assíncrona que simule uma pausa na execução do código (um "sono") pela quantidade de milissegundos informada antes de prosseguir.

* **Dificuldade:** Fácil
* **Conceito Chave:** Promises, Async/Await e Event Loop

---

### O que foi feito e Aprendizado
Para resolver este problema, criamos uma função que retorna uma **Promise**.
